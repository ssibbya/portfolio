Deploying a Static Website on AWS

Using S3 for Hosting and CloudFront for Fast Content Delivery

Step-by-Step Guide

1. Create an S3 Bucket

Go to AWS S3, create a bucket, enable static website hosting, and upload your HTML files.

2. Configure CloudFront

Create a CloudFront distribution, set the origin as your S3 bucket, and configure caching settings.

3. Update DNS with Route 53

Use AWS Route 53 to create a custom domain and point it to CloudFront.
