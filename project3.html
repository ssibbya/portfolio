CI/CD Pipeline for Static Website

Automate deployment using GitHub, CodePipeline and CodeDeploy.

Step-by-Step Implementation

1. Upload Static Website to GitHub

Create a GitHub repository and upload your static website files. Ensure that the repository is public or properly authenticated for CodePipeline access.

2. Create an S3 Bucket

Go to the AWS Console, navigate to S3, and create a bucket to store your website files. Enable static website hosting and set proper permissions.

3. Set Up CodePipeline

Navigate to AWS CodePipeline and create a new pipeline. Select GitHub as the source provider and connect it to your repository.

4. Deploy to S3 Using CodeDeploy

In the deploy stage of CodePipeline, select Amazon S3 as the deployment provider and choose your target S3 bucket.

5. Test Deployment

Push a change to your GitHub repository and check if the update is automatically deployed to S3.
